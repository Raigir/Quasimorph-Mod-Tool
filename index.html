Mod Workflow Tool

This app needs a local Node.js server to save files and manage your mod projects.
Easiest — Double-click the launcher
Windows: start.bat
Mac / Linux: start.sh
— or start manually —
Step 1 — Open a terminal in this folder
cd path/to/mod-workflow
Step 2 — Start the server
node server.js
Step 3 — Open in browser
http://localhost:8080
You're seeing this because the page was opened as a local file.
The app works fine once it's served through Node.js.
Don't have Node.js installed?
Download the LTS installer from nodejs.org — it's a normal installer, just click Next.
Mac: brew install node
Linux: sudo apt install nodejs
